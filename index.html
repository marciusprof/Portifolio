CCP Esportes

Nós somos a Escola Coronel Carlos Porto. Nossa escola é voltada para o público do Ensino Médio e somos uma escola PEI (Programa Ensino Integral). Como este Programa visa a formação integral dos estudantes, os esportes compõem nossa base, pois o esporte desenvolve valores que vão muito além das quadras, valores que atualmente são exigidos tanto para o mercado de trabalho quanto para a vida.
